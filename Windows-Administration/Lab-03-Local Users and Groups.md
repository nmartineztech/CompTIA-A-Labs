# Lab 05 – Local Users and Groups

## Objective

Practice creating and managing local Windows user accounts and assigning appropriate group memberships based on administrative requirements.

## Skills Practiced

* Local user account creation
* Initial password configuration
* Password policy settings
* User account management
* Local group membership
* Windows administrative tools
* User permissions
* Least privilege

## Environment

* Windows operating system
* Local Users and Groups management console
* Administrative account
* Local user accounts
* Local security groups
  
## Scenario

In this lab, I managed local Windows user accounts and group memberships. The lab required creating multiple user accounts, configuring initial passwords, requiring password changes at the next logon, and assigning each user to the appropriate security groups based on their required level of access.

## Tasks Performed

### 1. Created Local User Accounts

* Created three local user accounts.
* Configured an initial password for each account.
* Enabled the option requiring users to change their password at the next logon.
* Verified that the accounts were created successfully.

### 2. Configured Group Memberships

* Assigned the appropriate local security groups to each user account.
* Configured one user with administrative and standard user group membership.
* Configured one user with backup operator and standard user group membership.
* Configured one user with standard user group membership only.
* Verified that each account had only the required group memberships.

---

## Best Practices Followed

* Applied the principle of least privilege when assigning group memberships.
* Required users to change their initial passwords at first logon.
* Assigned users only to groups necessary for their responsibilities.
* Verified account and group configurations after making changes.
* Used built-in Windows administrative tools to manage local accounts.

## What I Learned

This lab provided hands-on experience managing local Windows user accounts and security groups. I learned how to create user accounts, configure initial password requirements, and assign appropriate group memberships. The lab also reinforced the importance of least privilege by ensuring users receive only the access required for their roles.
