# Lab 3 – Configure Desktop Settings

## Objective
Practice configuring Windows 11 desktop settings by managing language preferences and accessibility features for a specified user account.

## Skills Practiced
- Windows 11 desktop configuration
- User account settings
- Language pack configuration
- Accessibility settings
- High contrast accessibility schemes
- Windows Settings navigation
- Windows administration
  
## Environment

- Windows 11 operating system
- Simulated enterprise environment
- Specified user account
- Windows Settings application

---

## Scenario

In this lab, I configured Windows 11 desktop settings for a specified user account. The lab focused on enabling a required language pack and configuring accessibility features to meet user requirements.

## Tasks Performed

### 1. Configured Language Settings

- Accessed Windows 11 language and regional settings.
- Located the required language pack.
- Enabled the required language pack for the specified user account.
- Verified that the language configuration was successfully applied.

### 2. Configured Accessibility Settings

- Accessed Windows 11 accessibility settings.
- Located the available accessibility options.
- Enabled a high contrast accessibility scheme for the specified user account.
- Verified that the accessibility configuration was successfully applied.

## Best Practices Followed

- Configured settings for the correct user account.
- Verified that required Windows features were successfully enabled.
- Used built-in Windows 11 Settings to manage user preferences.
- Applied accessibility features based on user requirements.
- Confirmed successful configuration after completing each task.

## What I Learned

This lab provided hands-on experience configuring Windows 11 desktop settings based on specific user requirements. I learned how to enable language packs and configure accessibility features such as high contrast settings. The lab reinforced the importance of understanding Windows configuration options and verifying that user-specific settings are correctly applied.
