# Lab 05 – Format Removable Media

## Objective

Practice using Windows Disk Management to provision and format a removable storage device for use as a data storage medium.

## Skills Practiced

* Disk Management
* Disk provisioning
* Volume creation
* Disk formatting
* File system configuration
* Drive letter assignment
* Volume labeling
* Removable media management

## Environment
* Windows operating system
* Windows Disk Management utility
* File Explorer
* Removable storage device simulated as a fixed disk


## Scenario
In this lab, I provisioned a storage device for use as removable media. The task required creating a new volume using all available disk space, assigning a specific drive letter, formatting the volume with the FAT32 file system, and applying a volume label.

---

## Tasks Performed

### 1. Provisioned the Storage Device

* Opened the Windows **Disk Management** utility.
* Located the disk designated for removable media.
* Selected the unallocated space on the disk.
* Created a new simple volume.
* Configured the volume to use all available disk space.


### 2. Configured the New Volume

* Assigned the drive letter **U:**.
* Formatted the volume using the **FAT32** file system.
* Set the volume label to **USB**.
* Completed the volume creation and formatting process.


### 3. Verified the Configuration

* Opened **File Explorer**.
* Confirmed that the formatted volume appeared as **USB (U:)**.
* Verified that the storage device was accessible and correctly configured.

---

## Best Practices Followed

* Used Windows Disk Management to safely provision the storage device.
* Configured the volume according to the required specifications.
* Verified the drive letter and volume label after formatting.
* Confirmed the file system was configured correctly.
* Verified the completed configuration through File Explorer.

## What I Learned

This lab provided hands-on experience provisioning and formatting storage devices using Windows Disk Management. I learned how to create a new simple volume, assign a drive letter, select a file system, and configure a volume label. I also learned how to verify that a newly formatted storage device is accessible through File Explorer.
